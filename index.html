Workflow

Google Form to website in three clean steps

01

Athlete fills out one Google Form

Capture name, photo, position, class year, height, weight, GPA, Hudl, contact info, honors, and the athlete's "why" in one place.

02

Google Sheet becomes the master roster

Responses land in a sheet, and the included Apps Script turns them into a clean public athlete feed while keeping private contact fields out of the website.

03

Static site reads the athlete feed

Host the site free on GitHub Pages. Point it to the Apps Script JSON endpoint and your roster updates without rebuilding the website every time.
